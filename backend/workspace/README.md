# Sample Workspace

This folder is the project JakIDE opens by default (`PROJECT_ROOT`).
It contains small files across the supported languages so you can try the
editor, the AI assistant, and the terminal right away.

- `javascript/index.js` — Node script
- `python/app.py` — Python
- `go/main.go` — Go
- `react/Counter.tsx` — React + TypeScript
- `php/UserController.php` — Laravel-style controller
- `Dockerfile` — container build

Point the IDE at your own project by setting `PROJECT_ROOT` in `backend/.env`.
